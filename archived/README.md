# Archive

This directory contains archived files that are no longer actively maintained but may have historical value.

## Contents

### build-reports/
Point-in-time build status and completion reports from the development process:

- `*_COMPLETE.md` - Completion reports for various components (auth, backend, frontend, etc.)
- `*_STATUS.md` - Status snapshots at various points in development
- `*_ANALYSIS.md` - Technical analysis documents
- `CHANGELOG_*.md` - Change logs for specific features

These documents record the development history and may be useful for understanding why certain decisions were made.

### design-demos/
HTML prototypes of different dashboard UI concepts that were explored during design:

- `dashboard-v1-mission-control.html` - Dark, developer-focused design
- `dashboard-v2-journey-map.html` - Light, modern SaaS design
- `dashboard-v3-living-canvas.html` - Hybrid, adaptive design
- `option*.html` - Various color scheme and layout explorations

## Note

These files are not actively maintained. For current documentation, see the `docs/` directory in the project root.
